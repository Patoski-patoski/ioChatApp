How to use ioChatApp

  1. Sign up and login
  2. Add a friend registered on ioChatApp
  3. For first timers, a friend request will be sent to the email of the username
  4. This email contains your username and unique secret code between you and Friend
  5. The friend will accept the request by copying the code and your username and adding you
  6. Click "Chat" to start a conversation
  7. You can now send messages back and forth
